# /contribute-learnings

Push learnings to the central Lupio OS repository.

**Agent:** agents/learning-agent.md (Step 5)
**Action:** `bash .lupio/scripts/auto-contribute.sh`
**Requires:** `memory/prompt-changelog.md` or `memory/reusable-candidates.md` with new entries
**Output:** push to lupiodev/Lupio-os main, record in `memory/contributions.md`

**Token:** No file scanning — runs script directly.
